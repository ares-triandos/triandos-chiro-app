# Triandos Chiro App

Chiropractic & Acupuncture mobile app for iOS and Android.

## Setup

```bash
npm install --legacy-peer-deps
npm start
```

## Development

Run on iOS:
```bash
npm run ios
```

Run on Android:
```bash
npm run android
```

## Build

Build for production:
```bash
eas build --platform ios --profile production
eas build --platform android --profile production
```
